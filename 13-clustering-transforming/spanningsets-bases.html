13.3. Representing a Vector Using Projections: Spanning Sets and Bases#

Vector projection allows us to determine how to represent some portion of an arbitrary \(n\)-vector \(\mathbf{y}\) in terms of a reference \(\mathbf{x}\)-vector. Then given a sufficient set of reference \(n\)-vectors \(\mathbf{x}_0, \mathbf{x}_1, \ldots\), we should be able to reconstruct the original vector in terms of its vector projections \(\operatorname{proj}_{\mathbf{x}_0} \mathbf{y}, \operatorname{proj}_{\mathbf{x}_0} \mathbf{y}, \ldots\).

Let’s see how we can use this idea to represent a vector in terms of a rotated set of axes. We can represent the original \(x\) and \(y\) axes using unit vectors called the standard basis:

DEFINITION

standard basis:#

In a Euclidean vector space, the standard basis consists of a set of unique vectors whose components are all zeros except for a single 1. For the real plane, \(\mathbb{R}^2\), the standard basis is

\[\begin{equation*} \left\{ \mathbf{e}_x = \left[1, 0 \right], ~ \mathbf{e}_y = \left[0, 1 \right] \right\}, \end{equation*}\]

and for three-dimensional Euclidean space, \(\mathbb{R}^3\), the standard basis is \begin{equation*} \left{ \mathbf{e}_x = \left[1, 0, 0 \right], ~ \mathbf{e}_y = \left[0, 1, 0 \right], ~ \mathbf{e}_z = \left[0, 0, 1 \right] \right}. \end{equation*}

It is worth noting two important but trivial to prove properties of the vectors in any standard basis. Below, we show the properties for \(\mathbb{R}^3\) as an example.

  1. They are normal vectors: \(\| \mathbf{e}_x \| = \| \mathbf{e}_x \| = \| \mathbf{e}_z \| =1 \)

  2. They are mutually orthogonal, meaning that any pair of vectors is orthogonal. In \(\mathbb{R}^3\), \(\mathbf{e}_x \cdot \mathbf{e}_y =0\), \(\mathbf{e}_x \cdot \mathbf{e}_z =0,\) and \(\mathbf{e}_y \cdot \mathbf{e}_z =0.\)

Any set of vectors satisfying these two properties is called an orthonormal set:

DEFINITION

orthonormal set (of vectors)#

A set of vectors \(\left\{ \mathbf{x}_0, \mathbf{x}_1, \ldots, \mathbf{x}_{n-1}\right\}\) is an orthonormal set if and only if:

  1. The vectors are all normal; i.e., \(\|\mathbf{x}_i\|=1\) for all \(i = 0, 1, \ldots, n-1\).

  2. The vectors are mutually orthogonal; i.e., \(\mathbf{x}_i \cdot \mathbf{x}_j\) for all \(i \ne j\) in \(0, 1, \ldots, n-1\).

We can interpret the components of an arbitrary vector as the coefficients of a linear combination of the standard basis vectors. For example, if \(\mathbf{w} = [a,b]\), then \(\mathbf{w} = a \mathbf{e}_x + b \mathbf{e}_y\). For example, consider the vector \(\mathbf{w}=\left[2,1.5 \right]\). The plot below shows our usual plot of \(\mathbf{w}\) along with two vectors showing \(\mathbf{w} = 2 \mathbf{e}_x + 1.5\mathbf{e}_y.\)

Note also that the vectors \(2\mathbf{e}_x\) and \(1.5\mathbf{e}_y\) are the vector projections of \(\mathbf{w}\) onto \(\mathbf{e}_x\) and \(\mathbf{e}_y\):

import numpy as np
from numpy.linalg import norm

w = np.array([2, 1.5])
ex = np.array([1, 0])
ey = np.array([0, 1])

wpx = w@ex * ex / norm(ex)**2
print(f'Projection of z onto e_x = {wpx}')

wpy = w@ey * ey / norm(ey)**2
print(f'Projection of z onto e_y = {wpy}')

Projection of z onto e_x = [2. 0.]
Projection of z onto e_y = [0.  1.5]

Suppose our new axes are represented by unit vectors \(\mathbf{e}_{x,\theta}\) and \(\mathbf{e}_{y,\theta}\) to indicate that are rotated \(\theta\) degrees from the standard basis. The unit vectors that define the axes for a counter-clockwise rotation of \(60^\circ\) are shown below:

From trigonometry, we can see that the unit vectors for the axes are

\[\begin{align*} \mathbf{e}_{x,\theta} &= \left[~~~\cos \theta ~~ \sin \theta \right] \\ \mathbf{e}_{y,\theta} &= \left[~~~\cos \left(\theta + 90^\circ \right) ~~ \sin \left(\theta + 90^\circ \right)\right] \\ &= \left[ \,-\sin \theta~~ \cos \theta \right]. \end{align*}\]

Then for the particular case of \(60^{\circ}\), we define

theta_r = np.deg2rad(60)
ex60 = np.array([  np.cos(theta_r), np.sin(theta_r)])
ey60 = np.array([ -np.sin(theta_r), np.cos(theta_r)])

print(f'x-axis rotated by 60 deg CCW = {ex60}')
print(f'y-axis rotated by 60 deg CCW = {ey60}')
plotvec(ex60, ey60)
plt.xlim(-1,1);
plt.ylim(0,1);

x-axis rotated by 60 deg CCW = [0.5       0.8660254]
y-axis rotated by 60 deg CCW = [-0.8660254  0.5      ]

Note that rotating the axes does not change the fact that they are an orthonormal set. First, let’s check that they are still normal vectors:

norm(ex60), norm(ey60)

(1.0, 1.0)

Now let’s check that they are still orthogonal:

ex60 @ ey60

0.0

Now let’s find the projection of \(\mathbf{w}\) onto \(\mathbf{e}_{x,\theta}\) and \(\mathbf{e}_{y,\theta}\).

wpx=w@ex60
wpy=w@ey60

print(f'The scalar projections of w onto the rotated axes is {wpx,wpy}')

The scalar projections of w onto the rotated axes is (2.299038105676658, -0.982050807568877)

Now let’s visualize the vector projections onto these axes. The rotated axis vectors are shown as thin, black, non-transparent vectors, and the projections are shown as thicker, colored, and partially transparent vectors:

wx=wpx*xp
wy=wpy*yp
plotvec(xp, yp, width=0.005, colors=['k', 'k'])
plotvec(wx, wy, newfig=False, width=0.02,color_offset=3, alpha=0.7)
plt.xlim([-2,2])
plt.ylim([-1,2]);

The figure below shows two things:

  1. the original vector \(\mathbf{w}\), which is shown as a thin non-transparent vector, and

  2. the linear combination of the vector projections of \(\mathbf{w}\) onto the rotated axes, which is shown as a thicker, semi-transparent vector. The result shows that the linear combination of the vector projections completely reconstructs the original vector \(\mathbf{w}\).

plotvec(w, width=0.005, colors=['k'])
plotvec(wx+wy, newfig=False, width=0.02, color_offset=3, alpha=0.7)

We can confirm this by checking the norm of the error vector, which is the difference between the original vector and the sum of the vector projections:

e = w - (wx+wy)
print(f'The energy in the error vector is {norm(e):.3f}')

The energy in the error vector is 0.000

In the results above, we still show \(\mathbf{w}\) and the sum of the vector projections on the original axes. To translate \(\mathbf{w}\) to the rotated axes, we create a new vector \(\mathbf{w}_\theta\) for which the components are the scalar projections of \(\mathbf{w}\) onto the rotated axes:

w60=np.array([wpx,wpy])

If we plot \(\mathbf{w}_{60}\) and \(\mathbf{w}\) on the same axes, then \(\mathbf{w}_{60}\) is equivalent to a \(60^\circ\) clockwise rotation of \(\mathbf{w}\).

plotvec(w,w60, labels=['$\mathbf{w}$', '$\mathbf{w}_{60}$'])
plt.xlim([-1,3])
plt.ylim([-1.5,2])

(-1.5, 2.0)

We can confirm this using our formula for the angle between two vectors:

theta=np.arccos(w60@w/norm(w)/norm(w60))
print(f'theta = {np.rad2deg(theta): .2g}')

theta =  60

The representation of \(\mathbf{w}\) using the rotated axes has the same length as the original \(\mathbf{w}\):

print(f'Length of w = {norm(w):.2f}')
print(f'Length of w projected onto rotated axes = {norm(w60):.2f}')

Length of w = 2.50
Length of w projected onto rotated axes = 2.50

Any vector in \(\mathbb{R}^2\) can be represented using these new axes – we can simply rotate a vector by \(60^\circ\) to find its representation on the rotated axes.

We say that \(\mathcal{S}=\{\mathbf{x}', \mathbf{y}'\}\) is a spanning set for \(\mathbb{R}^2\) (or say that \(\mathcal{S}\) spans \(\mathbb{R}^2\)):

DEFINITION

spanning set#

A set of vectors \(\mathcal{S}= \left\{ \mathbf{x}_0, \mathbf{x}_1, \ldots, \mathbf{x}_{n-1}\right\}\) is a spanning set for some set of vectors \(\mathcal{Z}\) if every vector in \(\mathcal{Z}\) can be represented as a linear combination of the vectors in \(S\).

Note that we cannot remove either of the vectors from \(\mathcal{S}\) and still be able to represent everything in \(\mathbb{R}^2\). For example, we need both rotated axis vectors to represent \(\mathbb{z}\). We say that \(\mathcal{S}\) is minimal:

DEFINITION

minimal (spanning set)#

A spanning set \(\mathcal{S}\) for a set of vectors \(\mathcal{Z}\) is minimal if the removal of any member of \(\mathcal{S}\) would stop it from being a spanning set for \(\mathcal{Z}.\)

An equivalent condition for a spanning set to be minimal is that the vectors in the spanning set are linearly independent.

A minimal spanning set is also called a basis:

DEFINITION

basis (vector space)#

A set of vectors \(\mathcal{S}\) is a basis for a set of vectors \(\mathcal{Z}\) if \(\mathcal{S}\) is a minimal spanning set for \(\mathcal{Z}\).

The plural of basis is bases. We can always find a basis that is an orthonormal set, and such a basis is called an orthonormal basis:

DEFINITION

orthonormal basis (vector space)#

A set of vectors \(\mathcal{S}\) is an orthonormal basis for a set of vectors \(\mathcal{Z}\) if 1) \(\mathcal{S}\) is an orthonormal set, and 2) \(\mathcal{S}\) is a minimal spanning set for \(\mathcal{Z}\).

There are generally multiple bases for any set of vectors \(\mathbf{Z}\), but they all have the same cardinality, called the dimension of the set:

DEFINITION

dimension (vector space)#

Given a set of vectors \(\mathcal{Z}\) in a vector space, the dimension of \(\mathcal{Z}\), denoted \(\operatorname{dim} \mathcal{Z}\) is the cardinality of a basis for \(\mathcal{Z}\).

13.3.1. Applications of Alternative Bases#

Alternative bases are useful for many different purposes in data science and engineering:

  1. Alternative bases can be used when we want to reduce the dimensionality of a set of vectors. Instead of throwing away one element of a vector, we can find a basis that captures the most important features of a set of vectors in a small number of components in the alternate representation. This can be useful for:

    1. Plotting high-dimensional data by projecting the data onto a basis that reduces the data to 3 dimensions or fewer.

    2. Applying two-dimensional statistical techniques (like 2-D regression) to high-dimensional data.

    3. Reducing the complexity of statistical methods, signal processing techniques, or machine learning algorithms by eliminating redundant or irrelevant information.

    4. Performing data compression by preserving the most important information and discarding the least important information.

  2. Alternative bases can be used when we want to detect signals embedded in noise by projecting the noisy signal into those dimensions that enhance the signal components and suppress the noise.

  3. Alternative bases can be used to provide an alternative interpretation of data in terms of features such as frequency. In the next section, we show how an alternative basis constructed from complex sinusoids can be used to find the frequency content of time-domain vectors.
