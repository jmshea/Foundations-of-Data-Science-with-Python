11.5. Null Hypothesis Tests for Correlation#

If a correlation is observed, how can we know that it it statistically significant? Particularly in the case of small data, the correlation coefficient between two vectors is likely to not be close to zero just because of the randomness in the data. We can apply the concepts of null hypothesis significance testing (NHST) to determine whether an observed correlation is statistically significant. As usual, the first step in NHST is to establish a null hypothesis, \(H_0\). Since we are trying to assess whether the data is correlated, the null hypothesis is that the data is not correlated. We will consider two approaches: resampling and analytical techniques.

11.5.1. Resampling#

Consider data vectors for two features \(\mathbf{x}\) and \(\mathbf{y}\). Under resampling, we wish to generate new samples from the same distributions as \(\mathbf{x}\) and \(\mathbf{y}\) but without the data being correlated. We can understand how to draw data from these distributions with correlation in two ways:

  1. Any correlation in the data is because \(x\) and \(y\) vary together. If we draw bootstrap sample separately from \(\mathbf{x}\) and \(\mathbf{y}\), then the samples inherently do not vary together.

  2. Drawing bootstrap samples separately from \(\mathbf{x}\) and \(\mathbf{y}\) creates independent samples, and independent random variables are uncorrelated.

So to apply a resampling NHST:

  1. Determine a number of simulation iterations to be run, a significance (\(p\)-value) threshold \(\alpha\), and whether the test will be on-sided or two-sided.

  2. Initialize a counter to zero.

  3. In each simulation iteration:

    1. Use resampling (such as bootstrap) to separately draw vectors samples for each feature. The length of the vectors should be the same as the length of the original vectors \(\mathbf{x}\) and \(\mathbf{y}\).

    2. Calculate the correlation coefficient for the vectors.

    3. If the correlation coefficient is larger than the observed correlation increment the counter. Here larger can either refer to one particular direction (for a one-sided test) or magnitude (for a two-sided test.

  4. When all the simulation iterations are completed, divide the counter by the number of iterations to determine the \(p\)-value.

  5. Compare the \(p\)-value to \(\alpha\). If \(p \le \alpha\), then the result is statistically significant and the null hypothesis is rejected. If \(p > \alpha\), then the data is not sufficient to reject the null hypothesis.

Let’s demonstrate this using the Covid data set:

import pandas as pd

covid = pd.read_csv(
    "https://raw.githubusercontent.com/jmshea/Foundations-of-Data-Science-with-Python/main/03-first-data/covid-merged.csv"
)
covid.set_index("state", inplace=True)
covid["cases_norm"] = covid["cases"] / covid["population"] * 1000

urban =covid['urban'].to_numpy()
cases_norm = covid['cases_norm'].to_numpy()

As a reminder, the observed correlation is

import numpy as np

rho = np.corrcoef(urban,cases_norm)[0,1]
print(f'Observed correlation is rho = {rho:.2g}')

Observed correlation is rho = 0.42

Before we carry out our test, we need to determine our significance threshold and whether we are going to perform a one-sided or two-sided test. We will use a significance threshold of \(\alpha=0.05\), and we will carry out a two-sided test, which is more conservative than a one-sided test. The code to implement the two-sided NHST is shown below:

import numpy.random as npr

num_sims = 20_000

num_large_correlation = 0
for sim in range(num_sims):
  
  # Bootstrap resampling
  urban_sample = npr.choice(urban, len(urban))
  cases_norm_sample = npr.choice(cases_norm, len(cases_norm))
    
  # Two-sided test
  if abs(np.corrcoef(urban_sample, cases_norm_sample)[0,1]) > rho:
    num_large_correlation += 1
    
print(f'The probability of seeing a correlation'
      + f' as large as {rho: .2g}')
print(f'is approximately' +
      f'{num_large_correlation / num_sims : .2g}') 

The probability of seeing a correlation as large as  0.42
is approximately 0.0019

Since the \(p\)-value is much less than 0.05, the observed correlation is statistically significant.

11.5.2. Analytical Tests#

The mathematics behind the analytical NHST are beyond the scope of this book, but stats.linregress() returns a value called pvalue. This is the \(p\)-value for a two-sided NHST in which the null hypothesis is that the features are uncorrelated. The \(p\)-value can be shown to be approximated as the tail of a Student’s \(t\) distribution with \(n-2\) degrees of freedom.

For our example, the \(p\)-value is

import scipy.stats as stats
regress = stats.linregress(urban, cases_norm)

print(f'The p-value for a correlation NHST is {regress.pvalue : .2g}')

As expected, the \(p\)-values generated by bootstrap simulation and analysis are approximately equal.
